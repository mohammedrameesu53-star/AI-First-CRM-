# AI-First CRM HCP Module Assessment

A professional-grade, clean architecture implementation of an AI-First Healthcare Professional (HCP) CRM assessment system.

## Tech Stack
- **Frontend:** React, Redux Toolkit, Vanilla CSS, Lucide Icons, Vite
- **Backend:** FastAPI, LangGraph, Groq (gemma2-9b-it), PostgreSQL, SQLAlchemy

## Project Structure
- `backend/`: FastAPI server with clean architecture, database access, and LangGraph-powered AI agent.
- `frontend/`: React single page application with Redux Toolkit state management and clean modular CSS layout.
